# Prediksi Harga Saham BBRI Menggunakan LSTM vs XGBoost

**Judul:** Analisis Komparatif Performa Long Short-Term Memory (LSTM) dan eXtreme Gradient Boosting (XGBoost) dalam Prediksi Harga Saham BBRI (2015-2025)

**Target publikasi:** Sinta 2 / Sinta 3 atau Jurnal Ilmiah relevan

## Ringkasan

Penelitian ini membandingkan kinerja model deep learning **Long Short-Term Memory (LSTM)** dan model ensemble learning **eXtreme Gradient Boosting (XGBoost)** untuk memprediksi harga penutupan harian saham PT Bank Rakyat Indonesia (Persero) Tbk (BBRI) periode 2015-2025. Evaluasi dilakukan berdasarkan metrik akurasi regresi seperti **Root Mean Squared Error (RMSE)**, **Mean Absolute Error (MAE)**, dan **R-squared (R2)** untuk menemukan model yang paling optimal dalam menghadapi dinamika pasar modal Indonesia.
