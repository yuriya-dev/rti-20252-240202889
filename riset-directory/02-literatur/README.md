# 02-literatur

Kumpulan referensi & paper terkait untuk mendukung Tinjauan Pustaka (Tahap 5).

## Topik referensi yang relevan

- Prediksi harga saham dan analisis runtun waktu (time-series forecasting)
- Model Deep Learning khususnya Long Short-Term Memory (LSTM) untuk time-series
- Model Ensemble Learning khususnya eXtreme Gradient Boosting (XGBoost) untuk regresi
- Metrik evaluasi regresi (RMSE, MAE, MAPE, R2)
- Efisiensi pasar modal (Efficient Market Hypothesis) dan analisis volatilitas saham di Indonesia

## Isi yang diharapkan

- Salinan/ringkasan paper (PDF, catatan ringkasan, anotasi)
- Daftar pustaka terkumpul (format BibTeX/RIS) untuk diacu di [../07-manuskrip/](../07-manuskrip/)
