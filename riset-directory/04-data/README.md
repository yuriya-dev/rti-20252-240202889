# 04-data

Data mentah hasil pengujian — output dari **Tahap 1**, input untuk **Tahap 2 & 3**.

## Isi yang diharapkan

- Dataset mentah historis saham BBRI (BBRI.JK) berformat CSV.
- Dataset hasil pembersihan (clean dataset).
- Sequence dataset hasil preprocessing (MinMax scaler) yang siap digunakan untuk training model.

## Catatan

Data mentah di folder ini diunduh langsung dari Yahoo Finance API atau Investing.com untuk rentang waktu 2015-2025. Hasil evaluasi model dan prediksi disimpan di [../06-output/](../06-output/).
